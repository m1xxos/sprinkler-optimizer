# Map asset credits

Farm background images (`*.jpg`) and the tillable-tile data used to generate
`*.json` come from the **Stardew Planner** project by hpeinar:

- https://github.com/hpeinar/stardewplanner
- Licensed under the Apache License 2.0.

Stardew Valley is a game by ConcernedApe. Map art is derived from the game's
assets and is used here for a fan-made, non-commercial planning tool.

The `*.json` files are generated by `scripts/gen-maps.mjs` from the planner's
`*_tillable.json` layer data (16px tiles).
